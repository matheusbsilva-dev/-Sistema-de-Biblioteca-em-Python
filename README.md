# 📚 Sistema de Biblioteca em Python

## 📌 Descrição

Projeto simples desenvolvido em Python para gerenciamento de livros.
O sistema permite cadastrar e listar livros utilizando estruturas básicas como listas, dicionários e funções.

Este projeto foi criado com foco em prática de lógica de programação e organização de código.

---

## 🚀 Funcionalidades

* 📖 Adicionar novos livros
* 📚 Listar todos os livros cadastrados

---

## 🛠️ Tecnologias utilizadas

* Python 3

---

## ▶️ Como executar o projeto

1. Clone este repositório:

```
git clone https://github.com/seu-usuario/sistema-biblioteca-python
```

2. Acesse a pasta do projeto:

```
cd sistema-biblioteca-python
```

3. Execute o programa:

```
python main.py
```
